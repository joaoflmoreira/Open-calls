Sobre este calendário

Os dados foram recolhidos por pesquisa web no dia indicado em cima. Esta página é estática — para refrescar a lista com novos concursos e prazos, peça ao Claude "atualiza o calendário de open calls".

Como esta página vive numa sandbox local sem acesso aos sites das instituições, não é possível auto-refrescar diretamente. Para isso, podemos: (a) agendar uma tarefa semanal que peça ao Claude para reconstruir esta página; (b) adicionar os domínios das fontes em Settings → Capabilities, permitindo refresh em vivo.
